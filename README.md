	> 文件名: FMp3Play
	>				Fmp is Mp3 Player流行的递归名字很cool是吧．
	>					学c一个月零七天的练手之做．我会做好它!
	
	> 简介: win下的mp3播放工具．v1.1版用mci函数．
			v2.0用sdl2_mixer实现,v3.0使用bass音频库
	> 功能：可以单次和循环播放mp3和wav.
			这些功能将用参数形式在命令行下实现．
	> 初传时间：20171010
	
	> 使用方法：
		1.将快捷方式放在右键<发送到>菜单，直接把mp3发送到本工具上
		在命令行下输入命令:
		2.播放单曲:fmp xxx.mp3
		3.单曲循环:fmp r xxx.mp3
		4.指定单曲到最后一首:fmp p number
		5.指定序号单曲循环:fmp pr number
		6.显示当前目录歌曲列表:fmp ls
		7.播放全曲:fmp l
		8.关闭命令:fmp s
		
	注：放入windows文件夹全局使用更方便
